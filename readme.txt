== Description ==

* Visit [here](http://www.kahanit.com/kahanit-wp-fw) for documentation.
* Visit [here](http://www.kahanit.com/forums/forum/kahanit-wp-fw-1-x-x-qa) to post queries.
* Visit [here](http://www.kahanit.com/forums/forum/kahanit-wp-fw-1-x-x-feedback) to post feedback.
* Visit [here](http://www.kahanit.com/forums/forum/kahanit-wp-fw-1-x-x-bugs) to post bugs.

= Technical Requirement =
PHP Extension "pdo_mysql" must be loaded in your server

== Installation ==

1. Download and activate the plugin through the 'Plugins' menu in WordPress.
